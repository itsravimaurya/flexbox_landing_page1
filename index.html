Flexbox Landing Page

This landing page is created by Ravi Prakash Maurya to show the basic uses of flexbox. Flexbox is very important and useful tool in css.

Alignment & Space

A more efficient way to layout, align and distribute space among items in container.

Tricky Positioning

Flexbox usually solves tricky problems including how to position or dynamically resize elements on page.

Responsive Design

Flexbox makes building a website layout much, much easier and make it responsive.

Effortless Centering

Vertical and Horizontal centering is quite easy now. This was difficult before flexbox came along.

Simplified Layout

you don't need floats and positioning hacks. flexbox allows you to create you better layout with minimal code.

Intelligent Wrapping

Flex items can wrap onto new line automatically. No need to worry about overflowing content.
